    Class AbstractBuildStage<SubStage, Args>Abstract

    A configurable class for a single stage of a build system run via npm.

    To run a build stage, an instance of the class should be constructed. Then, call the async ['run'] | run method. The ['run'] | run method iterates through the ['subStages'] | subStages property and calls each substage as a method of this class (via the ['runStage'] | abstract runStage method).

    Example

    Type Parameters

    • SubStage extends string | never

      String literal of substage names to be run during this stage.

    • Args extends node.AbstractBuildStage.Args<SubStage>
