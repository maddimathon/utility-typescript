    Class AbstractBuildStage<SubStage, Args>Abstract

    A configurable class for a single stage of a build system run via npm.

    To run a build stage, an instance of the class should be constructed. Then, call the async run method. The run method iterates through the subStages property and calls each substage as a method of this class (via the abstract runSubStage method).

    Since

    0.4.2

    Type Parameters

    • SubStage extends string | never

      String literal of substage names to be run during this stage.

    • Args extends node.AbstractBuildStage.Args<SubStage>

      Argument object type for the build stage.
